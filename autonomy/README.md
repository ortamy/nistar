## 🎯 Философия

Никакого GitHub. Никакого Slack. Никакого Telegram. Никакого UptimeRobot.

Всё своё. Всё под контролем. Всё работает локально.

Эта папка содержит инструкции и скрипты для развёртывания полного цикла разработки и общения на собственном сервере. Ты не зависишь от облачных провайдеров, не платишь за подписки, не боишься блокировок и ограничений.

**Ключевые принципы:**
- Полный контроль над своими данными
- Отсутствие привязки к внешним сервисам
- Возможность работать без доступа в интернет (после первоначальной настройки)
- Прозрачность и понимание каждого компонента

---

## 📚 Что входит в экосистему

- **Gitea** — Git-сервер (альтернатива GitHub, GitLab)
- **Woodpecker CI** — CI/CD пайплайны (альтернатива GitHub Actions)
- **Mattermost** — командный чат (альтернатива Slack, Discord)
- **Ntfy** — Push-уведомления на телефон (альтернатива Pushover, Gotify)
- **Uptime Kuma** — мониторинг доступности сервисов (альтернатива Statuspage, UptimeRobot)

Все сервисы работают в Docker на одном сервере. Всё бесплатно. Всё с открытым исходным кодом.

## 🚀 Быстрый старт

Если у тебя уже есть сервер с Docker, просто выполни:

```bash
git clone https://github.com/your-username/nistar.git
cd nistar/autonomy
docker-compose -f docker-compose.yml up -d
```

После этого сервисы будут доступны по адресам:

· Gitea: http://IP_твоего_сервера:3000
· Mattermost: http://IP_твоего_сервера:8065
· Ntfy: http://IP_твоего_сервера:2586
· Uptime Kuma: http://IP_твоего_сервера:3001

Дальше следуй инструкциям для настройки интеграций.
