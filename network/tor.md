# 🧅 08. Tor и мосты — как не попасться

**Никто не знает, кто вы и откуда. Если всё настроено правильно.**

---

## 🕯️ Введение

Tor (The Onion Router) — это сеть, которая скрывает ваш IP-адрес, маршрутизируя трафик через несколько серверов (узлов) по всему миру. Каждый узел снимает один слой шифрования — как луковицу. Ни один узел не знает одновременно отправителя и получателя.

Ни провайдер. Ни сайт. Ни сотрудники Tor.

> Никто не знает, кто вы и откуда. Никто.

---

## 🎯 Как работает Tor

**Принцип**

Вы отправляете запрос. Он проходит через три случайных узла:

- Входной узел (знает ваш IP, но не знает, куда вы идёте)
- Промежуточный узел (не знает ни откуда, ни куда)
- Выходной узел (знает, куда вы идёте, но не знает, откуда)

**Выходной узел — слабое место**

Выходной узел может видеть ваш незашифрованный трафик (например, HTTP без HTTPS). Поэтому всегда используйте HTTPS. Ещё выходной узел может быть под контролем спецслужб (если они владеют узлом). Однако даже в этом случае они не увидят ваш IP — только то, что вы смотрели.

**Почему вас не видно**

- Входной узел не знает, что вы просматриваете
- Выходной узел не знает, кто вы
- Соединить эти данные невозможно (без контроля одновременно обоих узлов, что крайне сложно)

---

## 🚫 Почему Tor могут блокировать

**Как провайдер узнаёт об использовании Tor**

- Список публичных входных узлов Tor известен
- Провайдер видит, что вы подключаетесь к IP-адресу из списка
- Блокирует это подключение

**Результат:** вы не можете подключиться к сети Tor. Браузер пишет «не удаётся установить соединение».

**Как обойти:** мосты.

---

## 🌉 Мосты (Bridge) — обход блокировки

**Что это**

Секретные входные узлы Tor, которых нет в публичных списках. Провайдер не знает, что это узел Tor, и не блокирует.

**Где взять мосты**

- Встроенная в Tor Browser функция «Запросить мост»
- Отправить email на bridges@torproject.org
- Получить через Telegram-бота @GetBridgesBot

**Типы мостов**

- obfs4 — маскирует трафик под случайный шум (рекомендуется)
- Snowflake — прокси через пользователей браузера (запускаешь фрагмент кода на сайте, становишься мостом) — медленно, но работает даже в Китае
- WebTunnel — прячет Tor внутри WebSocket (экспериментально)

**Как настроить**

1. Открываете Tor Browser
2. Настройки → Мосты
3. Выберите «Использовать мост»
4. Скопируйте полученную строку моста
5. Подключитесь

## 🖥️ Tor Browser

**Это не просто браузер с поддержкой Tor. Это полностью изменённый Firefox.**

**Что изменено**

- Весь трафик идёт через Tor (по умолчанию)
- JavaScript ограничен (можно включить вручную)
- Запрещены трекеры, WebRTC, некоторые API
- Все вкладки изолированы друг от друга
- Блокируются сторонние cookie
- История не сохраняется

**Как использовать**

- Не открывайте документы в сторонних приложениях
- Не меняйте размер окна (уникальный размер — маячок)
- Не загружайте торренты (это не через Tor)
- Не входите в свои аккаунты (Google, Facebook, ВК), если не создавали их через Tor

---

## ⚠️ Чего нельзя делать в Tor

**Входить в старые аккаунты**

Зайдёте на Facebook через Tor, а Facebook запомнит ваш обычный IP. Теперь он знает, что «анонимный пользователь из Tor» — это вы.

**Правило:** используйте Tor только с аккаунтами, которые создали строго через Tor.

**Загружать торренты**

Торрент-клиенты не используют Tor и сломают всю анонимность. Ваш реальный IP будет виден в рое.

**Открывать PDF через сторонние программы**

PDF может содержать скрипты, которые отправят запрос вне Tor. Используйте встроенный просмотрщик Tor Browser.

**Включать JavaScript без необходимости**

Настройки Tor Browser: safest (отключает JS везде) — для параноиков. Standard — для обычных нужд, но с риском.

## 🔗 Tor onion-сайты (даркнет)

**Что это**

Сайты с доменом `.onion`. Доступны только через Tor.

**Преимущества перед обычными сайтами**

- Адрес — хэш ключа, не связан с IP
- Конец-конец шифрование (выходной узел не нужен)
- Не видны в публичных списках (если не распространять адрес)

**Где искать**

- The Hidden Wiki (оглавление, но осторожно — много мусора)
- Ahmia (поисковая система по .onion)

## 🔧 Как усилить анонимность в Tor

**❌ Выключен**

- Неправильно: просто запустить Tor Browser
- Риск: выходной узел видит ваш запрос

**✅ Базовый уровень**

- Открыть Tor Browser
- Включить мосты (obfs4)
- Выключить JavaScript
- Проверить на ipleak.net

**✅ Хорошо**

- Tor Browser
- Мосты
- VPN → Tor
- Tails OS

**✅ Лучше**

- Tails OS (загружена с флешки)
- Мосты + Tor
- Нет JavaScript
- Отключён WebRTC, IPv6, Flash
- Прокси на выходе (свой сервер)

**✅ Максимальная анонимность**

- Tails OS
- Мосты obfs4 или Snowflake
- VPN → Tor
- Использовать на чужом Wi-Fi
- Никогда не повторять сессии

---

## 🖥️ Tor + Tails (идеальное сочетание)

Tails OS заставляет весь трафик (все приложения) идти через Tor. Даже если приложение не использует прокси, в Tails оно пойдёт через Tor.

**Почему это лучше, чем Tor Browser на Windows**

- Windows шпионит за вами
- Даже если вы используете Tor Browser, операционная система может утекать данными
- Tails не оставляет следов и не знает о вас ничего

**Tails + мосты + VPN → Tor**

- Уровень анонимности: параноик
- Никто не узнает, кто вы
- Никто не узнает, что вы делаете
- Никакой суд не докажет

- Tor Browser + мосты (если блокируют)

**Хороший уровень**

- Tails OS + Tor Browser + мосты

**Максимальный уровень**

- Tails OS + VPN → Tor + мосты + обфускация
