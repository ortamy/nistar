# 🚀 10. Обход DPI — Shadowsocks, V2Ray, Hysteria

**Когда обычный VPN уже не спасает**

---

## 🕯️ Введение

Deep Packet Inspection (DPI) — технология, позволяющая провайдерам анализировать не только заголовки пакетов, но и их содержимое. DPI умеет определять WireGuard, OpenVPN, L2TP и даже Tor — и блокировать их. Даже если вы используете VPN, провайдер видит характерные паттерны и может оборвать соединение.

Для обхода DPI нужны протоколы, которые маскируют VPN-трафик под обычный HTTPS или случайный шум.

> VPN прячет содержимое. DPI смотрит на форму. Обход DPI меняет форму так, чтобы её нельзя было отличить от обычного интернета.

---

## 🎯 Что такое DPI

**Как работает**

Стандартный провайдер смотрит только на заголовки пакета (IP-адрес, порт, протокол). DPI заглядывает глубже:

- Анализирует TLS-рукопожатие (смотрит, какой сертификат, может ли это быть VPN)
- Вычисляет паттерны WireGuard, OpenVPN, Shadowsocks (если без обфускации)
- Проверяет размеры пакетов и частоту отправки

**Что умеет блокировать**

- OpenVPN (легко по характерному потоку данных)
- WireGuard (раньше было сложно, сейчас научились)
- Tor без мостов
- Shadowsocks без обфускации

**Что не умеет блокировать (пока)**

- Shadowsocks + obfs (маскировка под случайный трафик)
- V2Ray с WebSocket + TLS (маскировка под настоящий HTTPS)
- Hysteria (использует QUIC и имитирует видеозвонки)

---

## 🛡️ Shadowsocks

**Что это**

Лёгкий прокси-протокол (не VPN). Не шифрует весь трафик, только сокеты. Простой, быстрый. Был создан для обхода китайского файервола.

**Как обходит DPI**

- Трафик выглядит как случайный шум
- Стандартные сигнатуры не срабатывают

**Установка сервера**

```bash
apt install shadowsocks-libev
nano /etc/shadowsocks-libev/config.json
```

```json
{
    "server": "0.0.0.0",
    "server_port": 8388,
    "password": "your-password",
    "method": "chacha20-ietf-poly1305",
    "fast_open": true
}
```

```bash
systemctl enable shadowsocks-libev
systemctl start shadowsocks-libev
```

Установка клиента (Linux)

```bash
apt install shadowsocks-libev
ss-local -c /etc/shadowsocks-libev/config.json
```

Для маскировки (obfs)

```bash
apt install simple-obfs
obfs-server -c /etc/shadowsocks-libev/config.json --obfs http
```

Клиент подключается к obfs-порту, трафик маскируется под HTTP.

🔌 V2Ray / Xray

Что это

Платформа для создания прокси с множеством протоколов. Самый мощный инструмент для обхода DPI на сегодня.

Популярные протоколы

· VMess (основной, сложно детектится)
· VLESS (облегчённая версия VMess, без шифрования)
· Trojan (маскируется под HTTPS)
· Shadowsocks (встроен)

WebSocket + TLS (рекомендуется)

Трафик выглядит как обычный HTTPS:

1. Клиент подключается к серверу через TLS
2. Внутри TLS — WebSocket
3. Внутри WebSocket — VMess или VLESS
4. DPI видит обычный HTTPS и пропускает

Установка Xray

```bash
bash -c "$(curl -L https://github.com/XTLS/Xray-install/raw/main/install-release.sh)" @ install
```

Настройка сервера (config.json)

```json
{
  "inbounds": [
    {
      "port": 443,
      "protocol": "vless",
      "settings": {
        "clients": [{"id": "ваш-uuid"}],
        "decryption": "none"
      },
      "streamSettings": {
        "network": "ws",
        "security": "tls",
        "tlsSettings": {
          "certificates": [{"certificateFile": "/path/to/cert.pem", "keyFile": "/path/to/key.pem"}]
        },
        "wsSettings": {"path": "/ray"}
      }
    }
  ],
  "outbounds": [{"protocol": "freedom"}]
}
```

Клиент (Xray + VPN-приложение)

· V2RayNG (Android)
· Shadowrocket (iOS)
· Qv2ray (Linux, Windows)

⚡ Hysteria

Что это

Новый протокол, использующий QUIC (тот же транспорт, что в HTTP/3). QUIC сложно заблокировать, потому что он использует UDP и имитирует видеозвонки, стриминг.

Преимущества

· Очень быстрый
· Работает через UDP
· Маскируется под видеозвок (WebRTC)
· DPI не может отличить от обычного трафика

Установка сервера

```bash
wget https://github.com/HyNetwork/hysteria/releases/latest/download/hysteria-linux-amd64
chmod +x hysteria-linux-amd64
mv hysteria-linux-amd64 /usr/local/bin/hysteria
```

Настройка (config.json)

```json
{
  "listen": ":443",
  "cert": "/path/to/cert.pem",
  "key": "/path/to/key.pem",
  "obfs": "ваш-пароль-для-обфускации",
  "up_mbps": 100,
  "down_mbps": 100
}
```

Клиент (Hysteria)

```bash
hysteria -c client-config.json
```

📋 Сравнение протоколов

Shadowsocks

· Сложность настройки: низкая
· Обход DPI: средний
· Скорость: высокая
· Маскировка: низкая (без obfs)

Shadowsocks + obfs

· Сложность настройки: средняя
· Обход DPI: хороший
· Скорость: высокая
· Маскировка: средняя

V2Ray / Xray (VLESS+WS+TLS)

· Сложность настройки: высокая
· Обход DPI: отличный
· Скорость: средняя
· Маскировка: высокая (как HTTPS)

Hysteria

· Сложность настройки: высокая
· Обход DPI: отличный
· Скорость: очень высокая
· Маскировка: высокая (как видеозвонок)

---

🗺️ Когда что использовать

Базовый обход (если не блокируют сильно)

· Shadowsocks
· Просто, быстро
· Подходит для большинства случаев

Сильная блокировка (Китай, Турция, некоторые сети РФ)

· V2Ray (VLESS + WebSocket + TLS)
· Сложнее, но пройдёт почти везде

Для скорости (стриминг, торренты)

· Hysteria
· Очень быстро, но сложнее настраивать

Максимальная надёжность

· V2Ray + обфускация + CDN (Cloudflare)
· DPI видит только Cloudflare, ваш сервер скрыт
· Медленнее, но почти неблокируемо

После аренды

· Настроить сервер (Xray, Shadowsocks или Hysteria)

· Использовать вместо обычного VPN

---

🔒 Безопасность

Что делает прокси

· Скрывает ваш IP от сайтов
· Шифрует трафик от провайдера
· Обходит DPI

Чего не делает

· Не делает вас анонимным (сервер видит ваш IP)
· Не защищает от слежки после выхода с сервера (используйте HTTPS)

Как усилить анонимность

· Цепочка: клиент → прокси → Tor
· Сервер покупать за Monero
· Использовать выделенный VPS, на котором больше ничего не работает
